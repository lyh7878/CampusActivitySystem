# 校园活动报名与签到管理系统

## 项目背景
学校希望建设一套轻量级系统，支持活动发布、学生报名、名单审核、二维码签到和签到统计导出。

## 项目目标
- 支持活动管理员发布活动
- 支持学生在线报名
- 支持名单审核与查看
- 支持现场二维码签到
- 支持签到统计结果导出

## 目录结构说明
- `docs/`：所有项目文档
- `src/`：源代码（示例）
- `test/`：测试文件（示例）
- `assets/`：资源文件
- `README.md`：项目概况与协作约定

## 分支策略说明
- `main`：稳定受控分支，最终交付
- `dev`：开发集成分支
- `feature/*`：功能分支，如 `feature/wbs`

## 协作约定（单人模拟说明）
